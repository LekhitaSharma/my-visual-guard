Overview

I broke button padding in a side project and didn't notice for 2 weeks. Built this so it never happens again.

A lightweight framework that screenshots your UI in light/dark mode across mobile, tablet, and desktop — and fails CI if even 150 pixels change.

1. Capture

Playwright takes 6 parallel screenshots: 2 themes × 3 viewports.

2. Compare

Pillow computes pixel diff and saves a red diff image.

3. Block

GitHub Actions fails the PR and uploads diffs.
